# 🔄 전체 워크플로우

## 1️⃣ 음성 데이터 수집 & STT 변환

-   m4a → wav 변환
-   Google STT로 텍스트 추출
-   텍스트를 엑셀에 저장

## 2️⃣ AI 기반 상담 내용 분석 (RPA)

-   UiPath가 텍스트 로드 후 API 호출
-   AI가 제품명/주문일자/AS 여부 등 JSON 구조로 응답

## 3️⃣ 데이터 가공

-   JSON → DataTable 파싱
-   반품/AS 데이터 자동 분류 및 정제

## 4️⃣ 리포팅 자동화

-   Excel 저장
-   VBA로 자동 피벗 테이블 생성 및 시각화

# 🧑‍💻 주요 코드 예시

### 🔊 Python STT

``` python
def speech_to_text(audio_path):
    recognizer = sr.Recognizer()
    try:
        with sr.AudioFile(wav_path) as source:
            audio_data = recognizer.record(source)
            return recognizer.recognize_google(audio_data, language='ko-KR')
    except sr.UnknownValueError:
        return "음성을 인식할 수 없습니다."
```
